Definieren Sie URL's, welche vom System erlaubt werden. Sie haben dabei verschiedene Möglichkeiten, die URL zu definieren.

exact
Definieren Sie eine exakte URL, welche zugelassen wird. Dabei wird nur genau die definierte URL berücksichtigt.

starts with
Definieren Sie einen beliebigen Text. Fängt eine URL mit diesem Text an, wird sie zugelassen.
Bsp: www.lernstick.com
Wird zugelassen: www.lernstick.com/download

contains
Definieren Sie einen belieigen Text. Enthält eine URL diesen Text, wird sie zugelassen.
Bsp: lernstick
Wird zugelassen: www.lernstick.com Wird zugelassen: www.google.com/lernstick/info

custom
Definieren Sie einen Regulären Ausdruck. Matcht eine URL diesen Regex, wird sie zugelassen.
