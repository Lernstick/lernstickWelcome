Unterrichts-Software

Das Linux Advanced Teaching-System stellt eine einfache Klassenraumverwaltung bereit, inklusive:

  • Web-Sperren
  • Bildschirmsperren
  • Schüleransicht
  • Schülerzugriff
  • Gruppierung
  • Bidirektionale Dateikopien
  • Prüfungsumgebung

Tipp: Das Linux Advanced Teaching-System ist etwas anderes als die Lernstick-Prüfungsumgebung.

Ersteres ist eine Erweiterung für die offene Lernstick-Umgebung, Letzteres stellt eine komplett eigene, auf Prüfungen spezialisierte und abgesicherte Lernstick-Variante dar, die üblicherweise im Unterricht selbst nicht eingesetzt wird.

Wie bei allen Dingen im Leben gibt es bei beiden Varianten Vor- und Nachteile. Eine optimale Empfehlung, welche Variante verwendet werden sollte, kann nur unter Einbezug der Gegebenheiten am jeweiligen Standort getroffen werden.

Weitere Informationen zum Linux Advanced Teaching-System finden Sie auf dessen Webseite: http://teachingsystem.brg-kremszeile.ac.at
