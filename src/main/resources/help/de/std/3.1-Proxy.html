Proxy

Dieses Programm benötigt einen Internetzugang, um die ausgewählten zusätzlichen Programme herunterladen zu können.

In einigen Netzwerken müssen Sie einen Proxy verwenden, um auf das Internet zugreifen zu können. Hier können Sie die Proxy-Einstellungen festlegen, mit denen dieses Programm auf das Internet zugreifen kann.
