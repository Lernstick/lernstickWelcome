Proxy-Einstellungen

Dieses Programm benötigt einen funktionierenden Internetzugang, um die ausgewählten zusätzlichen Programme herunterladen zu können.

In einigen Netzwerken müssen Sie jedoch einen Proxy verwenden, um auf das Internet zugreifen zu können.

Hier können Sie die festlegen, ob ein Proxy für die Installation der Programme verwendet werden soll und dessen Zugangsdaten angeben.
