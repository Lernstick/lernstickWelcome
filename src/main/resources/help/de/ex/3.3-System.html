System

Töne werden üblicherweise vor der Ausgabe durch ein Soundsystem verarbeitet, das z.B. dafür sorgt, dass die gleichzeitige Tonausgabe von mehreren Programmen möglich wird oder dass die Ausgabe anstatt auf den Lautsprechern via Bluetooth-Kopfhörern erfolgt.

Leider gibt es noch immer Kombinationen von Hard- und Software (meistens betrifft es Programme, die via Wine auf älterer Hardware gestartet werden), bei denen die Verarbeitung durch das Soundsystem zu einer gestörten Tonausgabe führt.

Aus diesem Grund lässt sich hier der übliche Umweg der Tonausgabe über das Soundsystem deaktivieren. Allerdings muss dann auch auf den Komfort des Soundsystems verzichtet werden.
