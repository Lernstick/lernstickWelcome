Webseiten erlauben

Definieren Sie URL's, welche vom System erlaubt werden. Sie haben dabei verschiedene Möglichkeiten, die URL zu definieren.

Exakt
Definieren Sie eine exakte URL, welche zugelassen wird. Dabei wird nur genau die definierte URL berücksichtigt.
Beispiel: https://www.lernstick.com/logo.png

Beginnt mit
Definieren Sie einen beliebigen Text. Fängt eine URL mit diesem Text an, wird sie zugelassen.
Beispiel: https://www.lernstick.com
Wird zugelassen: https://www.lernstick.com/download

Enthält
Definieren Sie einen beliebigen Text. Enthält eine URL diesen Text, wird sie zugelassen.
Beispiel: lernstick
Wird zugelassen: https://www.lernstick.com
Wird zugelassen: https://www.google.com/lernstick/info

Erweitert
Definieren Sie einen regulären Ausdruck. Matcht eine URL diesen regulären Ausdruck, wird sie zugelassen.
