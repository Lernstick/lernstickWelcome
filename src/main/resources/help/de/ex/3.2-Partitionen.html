Partitionen

Im Menüpunkt “Partitionen” kann konfiguriert werden, ob bestimmte Aktionen nach dem Start des Lernsticks automatisch ausgeführt werden sollen, etwa das Lernstick-Willkommensprogramm starten oder den Informationsdialog, dass Änderungen am System beim nächsten Start wieder verworfen werden, anzuzeigen.
