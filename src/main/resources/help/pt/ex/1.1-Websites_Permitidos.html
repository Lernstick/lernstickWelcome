Websites permitidos

Aqui pode adicionar acessos a websites. Existem várias maneiras para definir um acesso:"

exacto
Definir o URL exacto que será permitido, por exemplo:
https://www.lernstick.com/logo.png

começa com
Definir uma string que deve coincidir com o início de um URL, a fim de permitir acessos ao mesmo."
exemplo: https://www.lernstick.com
permitido: https://www.lernstick.com/download

contém
Definir uma string que contenha o URL, a fim de permitir acessos ao mesmo."
exemplo: lernstick
permitido: https://www.lernstick.com
permitido: https://www.google.com/lernstick/info

personalizado
Definir uma expressão regular. Se o URL corresponder na expressão, será permitido."
