Hosts Permitidos

Aqui pode adicionar acessos a hosts. Existem várias maneiras para definir um acesso:"

Protocolo
Escolha entre o protocol UDP ou TCP.

Host
O nome do host ou endereço IP a ser permitido.

Porta ou intervalo de portas
A porta ou intervalo de portas a permitir separados com o caracter :
primeira porta:última porta, exemplo:
8000:8080

Descrição
Descrição do acesso.
