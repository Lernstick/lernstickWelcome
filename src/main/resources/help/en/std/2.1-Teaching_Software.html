Teaching Software

The Linux Advanced Teaching System provides a simple classrom management, including:

  • web blocking
  • screen locking
  • students view
  • students access
  • groups
  • bidirectional file copies
  • exam environment

Hint: The Linux Advanced Teaching System is different from the Lernstick exam environment.

The Linux Advanced Teaching System is an add-on for the open Lernstick learning environment, the Lernstick exam environment is a completely different environment, specialized and hardened for executing exams and is usually not used during class.

As with so many things in live, both variants have advantages and disadvantages. An optimal recommendation, which version to use, can only be given when all local circumstances are taken into account.

More information about the Linux Advanced Teaching System can be found on its website: http://teachingsystem.brg-kremszeile.ac.at
