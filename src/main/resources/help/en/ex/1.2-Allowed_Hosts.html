Allowed Hosts

You can configure the following options:

Protocol
Choose between UDP and TCP to be used as protocol.

Host
The host name or IP address to be whitelisted.

Port or Portrange
The port or port range of the host to be whitelisted. Port ranges must be separated with a colon:
first port:last port, e.g.:
8000:8080

Description
A description of the whitelist entry.
